# ACM VJIT Student Chapter Website

Welcome to the official repository for the **ACM VJIT Student Chapter** website. This platform serves as a central hub for all our activities, learning resources, event updates, and team information.

## 🚀 Features
- **Learning Paths**: Comprehensive roadmaps for various technologies (Web Dev, App Dev, Machine Learning, etc.).
- **Dev Tools & Cheatsheets**: Quick references and downloadable PDF guides (Git, Linux, Docker, SQL).
- **Events Gallery**: Highlights and updates from our recent workshops and sessions.
- **Team Directory**: Meet the passionate individuals behind our chapter's success.
- **Optimized & Accessible**: Built with modern web standards, optimized assets, and ARIA accessibility guidelines.

## 📂 Project Structure

## 🛠️ Local Development
To run this project locally, simply clone the repository and open `index.html` in your favorite web browser.
For testing features like local PDF loading, we recommend using a local development server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code.

```bash
# Clone the repository
git clone https://github.com/ayushkujha/ACM_VJIT.git

# Navigate into the project
cd ACM_VJIT
```
